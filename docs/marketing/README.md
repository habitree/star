## 빠른 요약

### 채널 우선순위
1. **SEO** (오가닉, 장기) — 854 정적 페이지 이미 구현
2. **SNS** (Instagram/TikTok) — 별자리 특성상 바이럴 가능성 최고
3. **네이버 검색광고** — 한국 핵심 트래픽 획득
4. **인플루언서** (나노/마이크로) — 초기 신뢰 구축
5. **이메일** — 리텐션 채널, LTV 극대화

### 예산 단계
- Phase 1 (월 50만원): 핵심 채널 테스트
- Phase 2 (월 200만원): 성과 채널 집중 투자
- Phase 3 (월 500만원): 스케일업 + 글로벌 진출
