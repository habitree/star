## 1. 커뮤니티 마케팅

### 1-1. 자체 커뮤니티 구축

**Discord 서버 운영**

```
서버명: ✨ 별자리 덕후 모임
채널 구성:
📢 공지사항
⭐ 오늘의-운세 (봇 자동 업데이트, 매일 07:00)
♈ 양자리방 | ♉ 황소자리방 | ... (12개 별자리 채널)
💑 궁합-질문
🔮 타로-리딩
🌙 달-위상-달력
💬 자유-수다방
🎁 이벤트-공지
👑 프리미엄-멤버 (프리미엄 구독자 전용)
```

봇 기능:
- `!운세 양자리` → 오늘 운세 자동 응답
- `!궁합 양자리 천칭자리` → 궁합 점수 응답
- 매일 07:00 자동 12별자리 운세 게시

Discord Nitro 부스트 인센티브:
- 서버 부스터 → 프리미엄 1개월 무료

**카카오 오픈채팅 운영**
- 채팅방명: "별자리 운세 & 타로" (참여자 1000명 목표)
- 매일 오전 운세 공유, 주간 이벤트 진행
- 방장: 서비스 운영팀 공식 계정

**네이버 카페 활용**
- 기존 대형 카페 내 정보 제공자로 활동
- 목표 카페: "별자리이야기", "타로마스터" 등 대형 카페
- 월 1회 공식 스폰서드 포스팅 협의

### 1-2. 커뮤니티 이벤트 캘린더

## 2. 이메일 마케팅

### 2-1. 이메일 수집 전략

**수집 포인트 (퍼널별)**

```
1. 온보딩 마지막 단계: "운세 알림 받기 (이메일)"
2. 운세 결과 페이지: "내일 운세 이메일로 받기"
3. 팝업: 30초 후 "매일 별자리 운세를 이메일로!"
4. 소셜 광고 리드폼: Meta Lead Ads
5. 콘텐츠 업그레이드: "PDF로 받기" (블로그 글 내)
6. 프리미엄 무료 체험: 이메일 필수 입력
```

**이메일 수집 목표**
- 월 1~3개월: 1,000명
- 월 4~6개월: 5,000명
- 월 7~12개월: 20,000명

### 2-2. 이메일 시퀀스 설계

**Welcome 시퀀스 (구독 직후 7일)**

| 발송 시점 | 제목 | 내용 | 목표 |
|---------|------|------|------|
| 즉시 | "✨ 오늘 [별자리] 운세 도착!" | 오늘 운세 전문 + 프리미엄 소개 | 첫 방문 유도 |
| D+1 | "어제 운세 맞았나요? 오늘은..." | 어제 피드백 + 오늘 운세 | 습관 형성 |
| D+3 | "[이름]님 별자리의 비밀 3가지" | 별자리 심층 성격 분석 | 관심 심화 |
| D+5 | "[이름]님과 가장 잘 맞는 별자리는?" | 궁합 기능 소개 | 기능 탐색 |
| D+7 | "7일 연속 운세 확인! 특별 배지 획득" | 스트릭 축하 + 프리미엄 업그레이드 CTA | 전환 유도 |

**일일 운세 뉴스레터**

```
발송 시간: 매일 오전 07:00 (현지 시간대)
제목 패턴: "🌟 [날짜] [별자리] 운세 - [오늘의 키워드]"
내용:
  - 오늘의 별자리 운세 요약 (3~5줄)
  - 오늘의 행운 키워드/색상/숫자
  - "자세한 운세 보기" CTA 버튼
  - 내일 운세 미리보기 티저 (자이가르닉 효과)
```

**재활성화 시퀀스 (7일 미방문)**

```
D+7 미방문: "보고 싶었어요 💙 오늘 [별자리] 운세"
D+14 미방문: "스트릭이 끊길 것 같아요... [별자리] 운세 확인"
D+30 미방문: "[별자리]자리에 큰 변화의 기운이... 지금 확인하세요"
```

## 4. 글로벌 마케팅 전략

### 4-1. 언어별 시장 전략

**한국어 (ko) - 핵심 시장**
- 네이버 SEO + 카카오 마케팅
- 한국형 점성술 + MBTI 연계 포지셔닝
- 인플루언서: 한국 운세/타로 유튜버

**영어 (en) - 글로벌 메인**
- Google SEO 집중 (경쟁 치열하나 볼륨 압도적)
- Reddit, Pinterest 커뮤니티 마케팅
- 미국/영국/호주 타임존 별 이메일 발송 최적화
- 목표: 전체 트래픽의 40% 이상

**일본어 (ja) - 우선 공략 해외 시장**
- 일본: 별자리 × 혈액형 결합 마케팅 포인트
- Twitter(X) 일본 사용자 많음 → 집중 공략
- 라인(LINE) 공유 버튼 추가
- Yahoo! Japan 검색 최적화 (일본 검색 2위)

**중국어 (zh) - 차별화 포지셔닝**
- 서양 별자리 (12궁) + 중국 전통 (12지신) 결합 콘텐츠
- Weibo, 小红书(샤오홍수) 계정 운영
- WeChat 미니프로그램 검토 (중장기)
- VPN 없이 접근 가능한 서비스 강점 부각

**스페인어 (es) - 장기 성장 시장**
- 중남미 시장: 인구 6억+, 별자리 관심 높음
- Instagram/TikTok 스페인어권 인플루언서 공략
- Google Ads 스페인어 키워드 (CPC 낮음 → ROI 유리)

### 5-2. ROI 시뮬레이션

**가정값**
```
평균 CPC: 150원
클릭 → 가입 전환율: 10%
가입 → 프리미엄 전환율: 3%
프리미엄 ARPU: 9,900원/월
광고 수익 ARPU (무료): 500원/월
LTV (12개월 기준): 프리미엄 90,000원 / 무료 3,000원
```

**Phase 1 시뮬레이션 (월 50만원 광고)**
```
총 클릭: 3,333명
신규 가입: 333명
프리미엄 전환: 10명
월 수익:
  - 프리미엄: 10 × 9,900 = 99,000원
  - 광고: 323 × 500 = 161,500원
  총: 260,500원
ROAS: 260,500 / 500,000 = 52% (초기 손실 예상)
```

**Phase 2 시뮬레이션 (누적 가입자 2000명, 월 200만원 광고)**
```
추가 클릭: 13,333명
추가 가입: 1,333명 (누적 3,333명)
프리미엄 사용자: 100명 (전환율 3%)
월 수익:
  - 프리미엄: 100 × 9,900 = 990,000원
  - 광고: 3,233 × 500/12 = 134,700원
  총: 1,124,700원
ROAS: 56% (손익분기점 접근)
```

**Phase 3 목표 (6개월, 누적 1만 가입자)**
```
프리미엄: 300명 × 9,900 = 2,970,000원
광고 수익: 9,700명 × 500/12 = 404,000원
총 월 수익: 3,374,000원
총 광고비: 500만원/월
손익분기: 월 수익 500만원 목표 = 프리미엄 505명 필요
```
