## 1. SNS 채널별 전략

### 1-1. Instagram / TikTok (핵심 채널)

**왜 인스타/틱톡인가?**
- 별자리 콘텐츠 해시태그 팔로워: `#horoscope` 5천만+, `#별자리` 3억+
- Z세대·MZ세대 = 주요 타겟 (18-35세 여성 70%)
- 짧은 영상/릴스로 바이럴 가능성 최고

**콘텐츠 유형**

| 포맷 | 내용 | 게시 주기 | 예상 도달 |
|------|------|---------|---------|
| 오늘의 별자리 카드 | 12별자리 × 1일 운세 카드 이미지 | 매일 오전 7시 | 500~2000/게시물 |
| 주간 운세 릴스 | 30초 영상, 이번주 별자리 하이라이트 | 매주 월요일 | 1000~5000 |
| "내 별자리 맞아?" 밈 | 별자리 성격 과장 밈 콘텐츠 | 주 3회 | 2000~10000 |
| Before/After 운세 | 실제 사용자 "맞춤운세" 반응 스크린샷 | 주 2회 | 500~3000 |
| 별자리 × 트렌드 결합 | 현재 트렌드(드라마/음악) + 별자리 | 트렌드 발생시 | 5000~50000 |

**TikTok 바이럴 공식**
```
훅(0~3초): "당신이 [별자리]라면 이거 무조건 공감"
중간(3~15초): 운세/성격 묘사 (바넘 효과 극대화)
CTA(15~30초): "오늘 [서비스명]에서 정확한 운세 확인하세요"
```

**해시태그 전략 (언어별)**
- 한국어: #별자리운세 #오늘의운세 #[별자리이름] #2026운세
- 영어: #horoscope #zodiac #astrology #dailyhoroscope #[sign]season
- 일본어: #星座運勢 #今日の運勢 #[星座名]座
- 중국어: #星座运势 #今日运势 #星座
- 스페인어: #horoscopo #signosdel zodiaco #astrologia

### 1-2. YouTube (중장기 SEO 채널)

**콘텐츠 포맷**

| 시리즈 | 길이 | 주기 | 키워드 타겟 |
|--------|------|------|-----------|
| 이번 주 별자리 운세 | 5~8분 | 주 1회 | "이번주 [별자리] 운세" |
| 별자리 성격 완전분석 | 10~15분 | 월 1회/별자리 | "[별자리] 성격 특징" |
| 별자리 궁합 시리즈 | 8~12분 | 월 2회 | "[별자리1] [별자리2] 궁합" |
| 수성역행 대처법 | 5분 | 역행 전 | "수성역행 2026" |
| 월간 타로 리딩 | 10분 | 월 1회 | "2026 [월] 별자리 운세" |

**YouTube Shorts 전략**
- 매일 30~60초 "오늘의 [별자리]" 쇼츠
- 썸네일: 별자리 심볼 + 강렬한 숫자("오늘 운세 98점!")
- 댓글 첫 고정: 서비스 링크 + "더 자세한 운세는 ↑"

## 2. 인플루언서 마케팅

### 2-1. 인플루언서 티어별 전략

| 티어 | 팔로워 | 접근법 | 예산/건 | 기대 효과 |
|------|--------|--------|--------|---------|
| 나노 (1K-10K) | 1,000~10,000 | 무료 체험 제공 + 리뷰 요청 | 0원 (서비스 제공) | 충성 팔로워 소구, 전환율 높음 |
| 마이크로 (10K-100K) | 10,000~100,000 | 협찬 + 수익쉐어 제안 | 10~50만원 | 타겟 커뮤니티 침투 |
| 매크로 (100K-1M) | 100,000~1,000,000 | 유료 스폰서십 | 100~500만원 | 브랜드 인지도 급상승 |
| 메가 (1M+) | 1,000,000+ | PR + 뉴스 가치 | 1000만원+ | 매스 인지도 (선택적) |

### 2-2. 핵심 타겟 인플루언서 카테고리

1. **별자리/타로/영성 크리에이터** (핵심)
   - 이미 팔로워가 관심 있는 타겟 → 전환율 최고
   - DM 접근: "저희 서비스 한번 써보시겠어요? 리뷰 부탁드립니다"

2. **MBTI/자기계발 크리에이터**
   - 별자리와 MBTI 결합 콘텐츠 기획 제안
   - "당신의 MBTI + 별자리 궁합 분석" 콘텐츠

3. **연애/관계 크리에이터**
   - 별자리 궁합 기능 강조
   - "별자리로 본 우리 관계 분석" 협업 콘텐츠

4. **일상 브이로그 크리에이터**
   - 자연스러운 일상에 운세 확인 습관 녹이기
   - "아침 루틴에서 별자리 확인" 노출

### 2-3. 어필리에이트 프로그램 설계

```
기본 구조:
- 인플루언서 고유 링크/쿠폰 발급
- 무료 사용자 유입: 건당 100원 (또는 무료)
- 프리미엄 전환: 건당 30% 수익 쉐어
- 월 정산, 최소 출금액 5만원
```

## 3. 바이럴 마케팅 전략

### 3-1. 심리적 바이럴 메커니즘

**A. 정체성 공유 욕구**
- "나는 [별자리]야" → 자신의 별자리 운세 공유 버튼
- 별자리별 특성 카드 → "이거 나 맞음ㅋㅋ" 공유 유도
- 구현: 각 운세 페이지에 "이미지로 저장" + "공유하기" 버튼

**B. 호기심 유발 (자이가르닉 효과)**
- "이번 주 [별자리]에 큰 변화가 올 수도..." → 친구에게 공유하게 만듦
- "당신의 별자리 친구는?"→ 궁합 결과 공유
- "내일 운세 먼저 보기" 티저 → SNS 공유시 해제

**C. 사회적 비교**
- "친구들의 별자리 운세도 확인" CTA
- 별자리별 점수 랭킹 → 자기 별자리 성적 공유
- "오늘 가장 운이 좋은 별자리는?" 공지 → 화제성

### 3-2. 바이럴 캠페인 기획

**캠페인 1: "내 별자리 챌린지"**
- 자신의 별자리 성격 특성 공유 → 태그 + 해시태그
- TikTok/릴스 형식, BGM 제공
- 상위 참여자 한 달 무료 프리미엄 증정

**캠페인 2: "별자리 궁합 테스트"**
- 연인/친구/가족과 궁합 결과 공유
- "우리 [X]%이나 맞아!" 이미지 카드 생성
- 공유할수록 추가 운세 포인트 지급

**캠페인 3: "올해의 별자리 예언"**
- 연초/생일에 1년 운세 예언 카드 생성
- "2026년 내 예언" 이미지 공유 → 연말에 맞춤 확인 유도

**캠페인 4: "별자리 MBTI 매핑"**
- 16 MBTI × 12 별자리 = 192 조합 콘텐츠
- "나는 INTJ 전갈자리" → 매우 구체적인 자기 정체성 표현
- 공유율 극히 높음 (정체성 콘텐츠)

### 3-3. 공유 유도 UI/UX

```
운세 결과 하단:
[ 📱 오늘 운세 공유하기 ]  [ 🖼️ 카드로 저장 ]  [ 🔗 링크 복사 ]

공유 이미지 자동 생성:
- 별자리 심볼 + 오늘 날짜
- 운세 핵심 한 줄 요약
- 점수 (예: 오늘 운세 87점)
- 서비스 URL 워터마크
```
