## 1. 검색광고 (Google Ads / Naver 검색광고)

### 1-1. Google Ads 전략

**캠페인 구조**

```
캠페인 1: 브랜드 보호
  └ 키워드: [서비스명], [서비스명] 운세
  └ 예산: 월 5만원 (경쟁사 차단 목적)

캠페인 2: 별자리 운세 (핵심)
  └ 광고그룹 1: 오늘 운세
    └ 키워드: 오늘 별자리 운세, 오늘의 운세, daily horoscope
  └ 광고그룹 2: 별자리별
    └ 키워드: 양자리 운세, 황소자리 운세, ... (12개 별자리)
  └ 광고그룹 3: 특수 운세
    └ 키워드: 별자리 궁합, 타로 운세, 생년월일 운세

캠페인 3: 시즌/이벤트
  └ 별자리 시즌 시작 전 2주 집중 집행
  └ 수성역행, 만월 등 천문 이벤트 연동
```

**광고 소재 A/B 테스트**
- 헤드라인 A: "오늘 당신의 별자리 운세는? 무료 확인"
- 헤드라인 B: "[별자리]자리 오늘 운세 - 연애·재물·건강 분석"
- 헤드라인 C: "매일 바뀌는 맞춤 별자리 운세 - 지금 무료 시작"

**입찰 전략**
- 초기: 수동 CPC (학습용)
- 3개월 후: Target CPA 전환 (목표 CPA: 가입 1건당 500원 이하)
- 스마트 입찰: Maximize Conversions → Target ROAS

**랜딩 페이지 최적화**
- 광고 → 별자리 선택 페이지 (장벽 최소화)
- 생년월일 입력 → 즉시 운세 결과 (3초 규칙)
- 유료 클릭 이탈률 목표: 40% 이하

### 1-3. 소셜 유료광고

**Meta Ads (Facebook/Instagram)**

오디언스 타겟팅:
```
핵심 타겟:
- 관심사: Astrology, Horoscope, Zodiac, Tarot
- 나이: 18-45세
- 성별: 여성 위주 (70%) + 남성 포함
- 행동: 영적/뉴에이지 앱 사용자

유사 타겟 (Lookalike):
- 현재 사용자 → 1% 유사 타겟 생성
- 높은 LTV 사용자 → 별도 1% 유사 타겟

리타겟팅:
- 웹사이트 방문자 (7일, 30일)
- 앱 설치 후 미활성 사용자
- 특정 별자리 페이지 방문자
```

광고 포맷:
- **스토리 광고**: 15초 별자리 운세 애니메이션
- **카드뉴스 광고**: 12별자리 운세 카드 슬라이드
- **리드 광고**: "지금 별자리 운세 무료로 받아보기" (이메일 수집)

**TikTok Ads**
- Spark Ads: 바이럴된 오가닉 영상에 유료 부스트
- In-Feed 광고: 별자리 운세 15~30초 영상
- TopView: 시즌 이벤트 시 집중 집행

## 2. SEO 전략

### 2-1. 기술적 SEO (현재 구현된 기능 강화)

**현재 상태**: 854 정적 페이지, x-default hreflang, JSON-LD, FAQSection 구현

**추가 개선 항목**

| 항목 | 현황 | 개선 방향 |
|------|------|---------|
| Core Web Vitals | 측정 필요 | LCP < 2.5s, CLS < 0.1, INP < 200ms |
| 이미지 최적화 | OG 이미지 미생성 | 별자리별 OG 이미지 12개 + default.jpg |
| 구조화 데이터 | NewsArticle, FAQ 적용 | BreadcrumbList, WebSite, Person 추가 |
| 내부 링크 | 부분 적용 | 별자리 간 교차 링크 강화 |
| 페이지 속도 | CF Workers 엣지 | 이미지 CDN 최적화 |

**E-E-A-T 강화**
- "저자" 섹션 추가: 점성술사 페르소나 프로필
- 콘텐츠 업데이트 날짜 명시
- 외부 학술 자료 인용 (점성술 근거)
- 사용자 리뷰/별점 표시 (Schema.org Review)

### 2-2. 키워드 전략 (다국어)

**한국어 타겟 키워드 (우선순위 순)**

| 클러스터 | 키워드 | 검색 의도 | 페이지 |
|---------|--------|---------|-------|
| 오늘 운세 | "오늘 별자리 운세", "오늘의 [별자리] 운세" | 즉시 | /ko/daily/[sign] |
| 별자리 정보 | "[별자리] 성격", "[별자리] 특징" | 정보 | /ko/zodiac/[sign] |
| 궁합 | "[별자리1] [별자리2] 궁합", "별자리 궁합" | 정보 | /ko/compatibility/[s1]/[s2] |

| 장기 | "[별자리] 2026 운세", "연간 별자리 운세" | 계획 | 별도 페이지 신설 필요 |

### 2-3. 콘텐츠 SEO 클러스터 전략

**Pillar Page + Cluster 구조**

```
Pillar: 별자리 운세 완전 가이드 (/ko/guide/zodiac-horoscope)
  ├── Cluster: 양자리 운세 (/ko/zodiac/aries)
  ├── Cluster: 황소자리 운세 (/ko/zodiac/taurus)
  ├── ... (12개 별자리)
  ├── Cluster: 별자리 궁합 가이드 (/ko/guide/compatibility)
  └── Cluster: 별자리 운세 보는 법 (/ko/guide/how-to-read)

Pillar: 오늘의 별자리 운세 (/ko/daily)
  ├── Cluster: 오늘 양자리 운세 (/ko/daily/aries)
  ├── ... (12개 별자리)
  └── Cluster: 이번 주 별자리 운세 (/ko/weekly)
```

**신설 필요 페이지**
1. `/[locale]/guide/` - 별자리 입문 가이드 (링크 허브)
2. `/[locale]/weekly/` - 주간 운세 (검색량 높음)
3. `/[locale]/yearly/2026/` - 2026년 별자리 운세 (시즌성 대형 트래픽)
4. `/[locale]/moon-phase/` - 달 위상 운세
5. `/[locale]/mercury-retrograde/` - 수성역행 가이드

### 3-2. 네이버 블로그 / 카카오 브런치 전략

**네이버 블로그 공략 (한국 SEO 핵심)**
- 네이버 검색 = 한국 1위 검색엔진, 블로그 콘텐츠 상위 노출
- 전략: 서비스 공식 블로그 운영 + 체험단 블로그 확산
- 게시 빈도: 최소 주 3회 (일상 운세 + 심층 콘텐츠)
- 체험단 운영: 별자리 덕후 블로거 20명 선발, 프리미엄 무료 + 글 작성 요청

**브런치 스토리 전략**
- "별자리 심리학" 시리즈 연재
- "점성술사의 하루" 에세이형 콘텐츠
- 인스타 감성 → 브런치 공유 → 네이버/구글 검색 유입
