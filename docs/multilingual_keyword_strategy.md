## 3. 롱테일 키워드 전략

### 3.1 롱테일 키워드의 중요성

| 특성 | 헤드 키워드 | 롱테일 키워드 |
|------|------------|--------------|
| 예시 | "horoscope" | "cancer horoscope today love" |
| 검색량 | 높음 | 낮음 |
| 경쟁도 | 높음 | 낮음 |
| 전환율 | 낮음 | 높음 |
| 의도 명확성 | 낮음 | 높음 |

### 3.2 롱테일 키워드 패턴

#### 패턴 1: [별자리] + [기간] + [카테고리]
```
영어: cancer weekly love horoscope
한국어: 게자리 이번주 연애운
중국어: 巨蟹座 本周 爱情运势
일본어: 蟹座 今週 恋愛運
```

#### 패턴 2: [별자리] + [별자리] + compatibility
```
영어: leo and virgo compatibility
한국어: 사자자리 처녀자리 궁합
중국어: 狮子座 处女座 配对
일본어: 獅子座 乙女座 相性
```

#### 패턴 3: [별자리] + personality/traits
```
영어: scorpio personality traits female
한국어: 전갈자리 여자 성격
중국어: 天蝎座 女生 性格
일본어: 蠍座 女性 性格
```

#### 패턴 4: [연도] + [별자리] + horoscope
```
영어: 2026 capricorn horoscope
한국어: 2026년 염소자리 운세
중국어: 2026年 摩羯座 运势
일본어: 2026年 山羊座 運勢
```

#### 패턴 5: how to / what is + [점성술 개념]
```
영어: how to calculate rising sign
한국어: 상승궁 계산하는 법
중국어: 如何计算上升星座
일본어: アセンダント 調べ方
```

#### 권장 URL 패턴

```
# 메인 페이지
/horoscope
/zodiac-signs
/birth-chart

/zodiac/[sign-name]

# 일일 운세
/horoscope/daily
/horoscope/daily/[sign-name]
예: /horoscope/daily/leo

# 주간/월간 운세
/horoscope/weekly/[sign-name]
/horoscope/monthly/[sign-name]

# 궁합
/compatibility
/compatibility/[sign1]-[sign2]
예: /compatibility/leo-virgo

# 출생 차트
/birth-chart
/birth-chart/calculator

# 점성술 가이드
```
