# 서비스 고도화 기획자료 — 파트별 진행

이 폴더는 **컨텍스트 길이를 고려해** 고도화 기획자료를 **파트별로** 작업할 수 있도록 나눈 문서와, 진행 계획·프롬프트를 담고 있습니다.

## 시작하기

1. **전체 흐름 파악**: [00_전체_계획_및_진행_가이드.md](00_전체_계획_및_진행_가이드.md)  
   - 진행 순서, 파트별 입·출력, 맥락 이어짐 방식 정리
2. **파트 실행**: [01_파트별_프롬프트_문서.md](01_파트별_프롬프트_문서.md)  
   - Part A → B → C → D → 부록 순으로, 복사용 프롬프트로 지시
3. **작업 대상**: `part_a_*.md` ~ `part_d_*.md`, `part_부록.md`  
   - 한 번에 하나의 파트 문서만 열어 작업하면 컨텍스트를 줄일 수 있음
