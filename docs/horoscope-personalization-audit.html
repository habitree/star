1. 현재 시스템 — "같은 별자리 = 동일 운세" 증명

현재 시드 공식

seed = year × 10,000,000 + month × 100,000 + day × 1,000 + signNum × 10 + catHash

시드를 구성하는 변수: 오늘 날짜 + 별자리 번호 + 카테고리

⚠️ 생년월일(birthDate)은 별자리 분류에만 사용 — 시드 계산에 미포함

구체적 수치 예시 (2026-03-07, 황소자리, 연애운)

사용자생년월일별자리시드값점수결과
사용자 A1990-05-01황소자리 20260307 × 연산 → 202,603,072,020 + hash 67점 SAME
사용자 B1995-05-15황소자리 동일한 날짜 + 동일한 signNum(2) → 동일 시드 67점 SAME
사용자 C2000-04-28황소자리 → 동일 시드 67점 SAME

* 황소자리 사용자는 전 세계 모두 동일한 점수·텍스트·타로 카드를 받습니다.

2. 새 시스템 설계

개인화 시드 공식 (2-레이어)

Layer 1 — 생년월일 해싱

// 생년월일 → 32비트 분산 값 (Knuth 곱셈 해싱)
const personalizer = (birthYear - 1900) × 366 + birthMonth × 31 + birthDay;
const mixed = Math.imul(personalizer, 2654435761) >>> 0;  // Knuth constant
seed = (baseSeed XOR mixed) >>> 0;

Layer 2 — 출생 시간 (선택적)

// 출생시간 → 12 어센던트 구간 (2시간 단위)
if (birthTime) {
  const segment = Math.floor((hours × 60 + minutes) / 120);  // 0 ~ 11
  seed = (seed + Math.imul(segment, 7919)) >>> 0;  // 소수 혼합
}

// 브라우저 콘솔에서 실행 — 개인화 효과 검증
const today = new Date('2026-03-07');

const h1 = generateDailyHoroscope('taurus', today, 'ko', '1990-05-01');
const h2 = generateDailyHoroscope('taurus', today, 'ko', '1995-05-15');

// 개인화 검증: true 이어야 함
console.assert(
  (h1.overall as any).detailedScore !== (h2.overall as any).detailedScore,
  '❌ 개인화 실패: 두 사용자가 동일한 점수'
);

// 결정론 검증: true 이어야 함
const h1_again = generateDailyHoroscope('taurus', today, 'ko', '1990-05-01');

  (h1.overall as any).detailedScore === (h1_again.overall as any).detailedScore,
  '❌ 결정론 실패: 동일 입력에서 다른 결과'
);
