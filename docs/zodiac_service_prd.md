## 1. Executive Summary

### 프로젝트 개요
글로벌 점성술 시장의 급격한 성장과 MZ세대의 자기이해/정체성 탐구 트렌드에 맞춰, AI 기반 개인화 별자리 분석 서비스를 구축합니다.

### 핵심 가치 제안
- **정확한 천문학적 데이터** 기반 출생 차트 분석
- **AI 개인화**를 통한 맞춤형 운세 및 조언
- **다국어 지원**으로 글로벌 사용자 확보
- **심리학적 원리**를 활용한 높은 사용자 몰입도

### 목표 지표 (MVP 6개월 후)
| 지표 | 목표 |
|------|------|
| MAU | 50만 명 |
| DAU/MAU 비율 | 30% 이상 |
| 평균 세션 시간 | 5분 |
| 유료 전환율 | 3% |
| 30일 리텐션 | 25% |

## 2. 시장 분석

### 2.1 시장 규모
| 연도 | 글로벌 시장 규모 | 성장률 |
|------|-----------------|--------|
| 2024 | $3.94B | - |
| 2025 | $4.75B | +20.6% |
| 2031 | $22.8B (예상) | CAGR 25%+ |

- **모바일 앱 비중**: 전체 시장의 60% 이상
- **2025년 앱 수익 예상**: $2.85B

### 2.2 사용자 통계 (Harris Poll 2024)
- **95%**가 자신의 별자리를 알고 있음
- **70%**가 점성술을 믿음
- **85%**가 점성술에 긍정적/중립적 태도
- 주요 이용 목적: 오락(58%), 자기이해(31%)

### 2.3 SWOT 분석

#### Strengths (강점)
- AI 기반 개인화 분석 기술
- 다국어 지원 역량
- 심리학적 마케팅 전문성
- 빠른 개발 사이클

#### Weaknesses (약점)12
- 신규 브랜드로 인지도 부족
- 초기 콘텐츠 양 제한
- 점성술 전문가 네트워크 부재

#### Opportunities (기회)
- MZ세대 자기탐구 트렌드
- 웰니스/마인드풀니스 시장 성장
- AI 기술 발전으로 개인화 품질 향상
- 글로벌 진출 용이성

#### Threats (위협)
- Co-Star, 포스텔러 등 기존 강자
- 무료 서비스 범람
- 점성술에 대한 부정적 인식
- 플랫폼 정책 변화 (앱스토어 등)

## 3. 타겟 사용자 & 페르소나

### 3.1 주요 타겟 세그먼트

| 세그먼트 | 연령 | 특성 | 주요 니즈 |
|----------|------|------|----------|
| **Gen Z 탐험가** | 18-25 | 자기정체성 탐구, SNS 활발 | 재미, 공유 가능한 콘텐츠 |
| **밀레니얼 전문가** | 26-35 | 커리어/연애 고민, 웰니스 관심 | 의사결정 참고, 스트레스 해소 |
| **라이트 유저** | 전 연령 | 가끔 재미로 확인 | 빠른 일일 운세, 무료 콘텐츠 |
| **딥 엔쓰지스트** | 25-45 | 점성술 깊이 관심 | 상세 차트 분석, 전문 콘텐츠 |

### 3.2 주요 페르소나

#### 페르소나 1: 지민 (24세, 대학생)
- **배경**: 취업 준비 중, 인스타그램 활발
- **목표**: 매일 가벼운 운세 확인, 친구들과 궁합 테스트
- **고민**: 미래에 대한 불안, 결정 어려움
- **행동 패턴**: 아침 출근길 앱 확인, 결과 스토리 공유
- **가치 기대**: 무료, 예쁜 디자인, 공유 용이

#### 페르소나 2: 수현 (32세, 마케터)
- **배경**: 경력 5년차, 연애 중, 워라밸 중시
- **목표**: 주요 결정 시 참고, 연인과의 궁합 확인
- **고민**: 이직 고민, 관계 발전 방향
- **행동 패턴**: 주 2-3회 접속, 월간 운세 정독
- **가치 기대**: 정확한 분석, 구체적 조언, 프라이버시

#### 페르소나 3: 마이클 (28세, 프리랜서, 미국)
- **배경**: 점성술에 진지한 관심, 여러 앱 사용 경험
- **목표**: 정확한 출생 차트, 트랜짓 분석
- **고민**: 기존 앱들의 피상적 분석에 불만
- **행동 패턴**: 매일 접속, 깊이 있는 콘텐츠 탐색
- **가치 기대**: 전문성, 정확성, 유료 가치

## 4. 핵심 기능 명세

### 4.1 기능 우선순위 매트릭스

| 우선순위 | 기능 | 임팩트 | 난이도 | MVP 포함 |
|----------|------|--------|--------|----------|
| P0 | 일일/주간/월간 운세 | 높음 | 낮음 | ✅ |
| P0 | 12별자리 상세 페이지 | 높음 | 낮음 | ✅ |
| P0 | 별자리 궁합 | 높음 | 중 | ✅ |
| P1 | 출생 차트 생성 | 높음 | 높음 | ✅ |
| P1 | 사용자 인증/프로필 | 중 | 중 | ✅ |
| P2 | AI 개인화 분석 | 높음 | 높음 | ❌ |
| P2 | 푸시 알림 | 중 | 낮음 | ❌ |
| P2 | 소셜 공유 | 중 | 낮음 | ❌ |
| P3 | 커뮤니티 | 중 | 높음 | ❌ |
| P3 | 1:1 상담 연결 | 높음 | 높음 | ❌ |

### 4.2 MVP 기능 상세

#### 4.2.1 일일/주간/월간 운세
**설명**: 12별자리별 시기별 운세 제공

**기능 요구사항**:
- 일일 운세: 매일 오전 6시 업데이트
- 주간 운세: 매주 월요일 업데이트
- 월간 운세: 매월 1일 업데이트
- 카테고리: 종합운, 연애운, 금전운, 건강운, 직장운

**기술 요구사항**:
- 운세 콘텐츠 CMS 연동
- 캐싱 전략 (CDN 활용)
- 오프라인 모드 지원

#### 4.2.2 별자리 상세 페이지
**설명**: 12별자리별 종합 정보 페이지

**포함 정보**:
- 기본 정보 (기간, 원소, 행성, 심볼)
- 성격 특성 (장점/단점)
- 연애 스타일
- 직업 적성
- 유명인 예시
- 상성 좋은/안 좋은 별자리

#### 4.2.3 별자리 궁합
**설명**: 두 별자리 간 궁합 분석

**기능 요구사항**:
- 연애 궁합, 친구 궁합, 비즈니스 궁합
- 궁합 점수 (100점 만점)
- 장점/주의점 텍스트
- 조합별 144개 (12x12) 콘텐츠

#### 4.2.4 출생 차트 생성
**설명**: 정확한 출생 정보 기반 천체 배치도

**입력 정보**:
- 생년월일 (필수)
- 출생 시간 (선택, 정확도 향상)
- 출생 장소 (필수)

**출력 정보**:
- 태양 별자리 (Sun Sign)
- 달 별자리 (Moon Sign)
- 상승궁 (Rising Sign)
- 각 행성 위치
- 하우스 배치
- 시각적 차트 이미지

**기술 요구사항**:
- Swiss Ephemeris 라이브러리 활용
- 시간대/위도경도 변환
- 차트 렌더링 라이브러리

### 4.3 Phase 2 기능

#### AI 개인화 분석
- 사용자 출생 차트 기반 맞춤 운세
- 현재 행성 트랜짓과 개인 차트 비교 분석
- GPT 기반 자연어 해석 생성

#### 푸시 알림
- 일일 운세 알림 (사용자 설정 시간)
- 중요 천문 이벤트 알림 (수성 역행 등)
- 재방문 유도 알림

#### 소셜 기능
- 운세 결과 카드 이미지 생성
- SNS 공유 (인스타 스토리, 트위터)
- 친구 초대 리워드

## 6. 기술 아키텍처 (개요)

### 6.1 시스템 구성

```
┌─────────────────────────────────────────────────────┐
│                    Frontend                         │
├────────────────────┬────────────────────────────────┤
│   Web (Next.js)    │   Mobile (React Native)        │
└────────────────────┴────────────────────────────────┘
                          │
                          ▼
┌─────────────────────────────────────────────────────┐
│                   API Gateway                        │
│              (Rate Limiting, Auth)                   │
└─────────────────────────────────────────────────────┘
                          │
          ┌───────────────┼───────────────┐
          ▼               ▼               ▼
┌──────────────┐ ┌──────────────┐ ┌──────────────┐
│ Content API  │ │ Astrology API│ │  User API    │
│  (운세/콘텐츠) │ │  (차트 계산)  │ │  (인증/프로필) │
└──────────────┘ └──────────────┘ └──────────────┘
          │               │               │
          ▼               ▼               ▼
┌─────────────────────────────────────────────────────┐
│                    Database                          │
│   PostgreSQL (사용자) + MongoDB (콘텐츠) + Redis      │
└─────────────────────────────────────────────────────┘
```

### 6.2 핵심 기술 스택
- **Frontend**: Next.js (Web), React Native (Mobile)
- **Backend**: Node.js + Express / FastAPI
- **Database**: PostgreSQL, MongoDB, Redis
- **Astrology Engine**: Swiss Ephemeris
- **AI**: OpenAI GPT-4 API
- **Infra**: AWS (ECS, RDS, S3, CloudFront)
- **Analytics**: Mixpanel, Google Analytics
