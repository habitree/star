STEP 1 — Ko-fi 팁 버튼 설정

수수료 0~5% · 계정 생성 즉시 사용 · 결제 시스템 불필요
1

Ko-fi 계정 생성

아래 링크에서 Ko-fi 계정을 만드세요. Google 계정으로도 즉시 가입 가능합니다.

🔗 ko-fi.com 바로가기 →
  • 우측 상단 Sign Up 클릭
  • 구글 또는 이메일로 가입
  • Username 설정: starzodiac 입력 시도 (이미 사용 중이면 다른 이름 선택)
⚠️ Username이 starzodiac이 아닌 경우, 아래 코드 2곳의 URL을 교체해야 합니다.
2

프로필 설정

  • Page Name: 별자리 운세 (Star Zodiac)
  • Goal: "서비스 운영비 지원" 또는 "Keep the Stars Shining"
  • Profile Image: 서비스 로고 업로드
  • Payment: PayPal 또는 Stripe 연결 (수익 수령용)
ℹ️ PayPal 계정이 없으면 paypal.com에서 먼저 개설하세요. 한국 계좌 연결 가능합니다.
3

코드에서 URL 확인 또는 교체

Username이 starzodiac과 다른 경우 아래 두 파일을 수정하세요.

📄 src/components/horoscope/PersonalizedResult.tsx (약 417번째 줄)
- href="https://ko-fi.com/starzodiac"
+ href="https://ko-fi.com/YOUR_USERNAME"
📄 src/components/layout/Footer.tsx (약 81번째 줄)
- href="https://ko-fi.com/starzodiac"
+ href="https://ko-fi.com/YOUR_USERNAME"
4

교체 후 빌드 확인

# URL 교체 후 빌드 테스트
npm run build
✅ 완료 체크리스트
🌙

STEP 2 — Moon Reading 어필리에이트

판매당 75% 수익 · 타로 카드 섹션과 자연스럽게 연결
1

어필리에이트 신청

Moon Reading 공식 어필리에이트 페이지에서 신청합니다.

🔗 moonreading.com/affiliates 바로가기 →
  • 이름, 이메일, 웹사이트 URL (https://star-zodiac.pages.dev) 입력
  • 프로모션 방법: "별자리 운세 서비스의 타로 카드 섹션 연계" 설명
  • 승인까지 보통 1~3 영업일 소요
ℹ️ 대안: ClickBank 또는 ShareASale 통해서도 Moon Reading 프로그램에 참여 가능합니다.
→ clickbank.com 검색창에 "moon reading" 검색
2

승인 후 — 어필리에이트 링크 취득

승인 이메일에 포함된 개인 추적 링크를 확인하세요. 보통 형식은 다음과 같습니다:

https://moonreading.com/?ref=YOUR_AFFILIATE_ID
또는 ClickBank 링크:
https://hop.clickbank.net/?vendor=moonread&affiliate=YOUR_ID
3

코드에서 URL 교체

📄 src/lib/affiliate-config.ts — moon-reading.url
- url: 'https://moonreading.com',
+ url: 'https://moonreading.com/?ref=YOUR_AFFILIATE_ID',
⚠️ TODO 주석(// TODO: 신청 후 실제 어필리에이트 URL로 교체)을 찾아 위와 같이 교체하세요.
4

빌드 및 배포

npm run build
npm run cf:deploy
✅ 완료 체크리스트
🔮

STEP 3 — LifeReader & Keen 어필리에이트

LifeReader $80/고객 · Keen $100/고객 · 궁합 페이지 연계

🔮 LifeReader

1

LifeReader 어필리에이트 신청

🔗 lifereader.com/affiliates 바로가기 →
  • 웹사이트 URL, 월 방문자 수 (예상치 입력 가능), 트래픽 소스 설명
  • 트래픽 소스: "별자리/운세 서비스 — 궁합 결과 페이지 노출" 입력
  • 승인까지 2~5 영업일 소요
2

승인 후 코드 교체 (LifeReader)

📄 src/lib/affiliate-config.ts — lifereader.url
- url: 'https://www.lifereader.com',
+ url: 'https://www.lifereader.com/?aff=YOUR_ID',

⭐ Keen Psychic

1

Keen 어필리에이트 신청 (Commission Junction 경유)

Keen은 CJ Affiliate (Commission Junction)를 통해 운영됩니다.

🔗 cj.com 회원가입 →
  • CJ 계정 생성 → 퍼블리셔(Publisher)로 등록
  • 검색창에 "Keen" 검색 → 프로그램 신청
  • 신청 후 Keen 측 승인까지 1~7일 소요
ℹ️ 대안: ShareASale에서도 유사한 Psychic/Astrology 어필리에이트 프로그램 다수 검색 가능합니다.
2

승인 후 코드 교체 (Keen)

📄 src/lib/affiliate-config.ts — keen.url
- url: 'https://www.keen.com',
+ url: 'https://www.keen.com/?utm_source=affiliate&utm_medium=YOUR_CJ_ID',
✅ 3개 URL 모두 교체 후: npm run build && npm run cf:deploy

STEP 4 — Ezoic 광고 네트워크 전환

AdSense 대비 RPM 3~5배 · 트래픽 제한 없음 · AI 광고 최적화
⚠️ Ezoic 설정은 DNS 레코드 변경이 포함되어 가장 복잡합니다. 단계를 순서대로 따라가세요. Cloudflare를 사용 중이므로 Cloudflare 연동 방식을 권장합니다.
1

Ezoic 계정 신청

🔗 ezoic.com 바로가기 →
  • 우측 상단 Get Started 클릭
  • 웹사이트 URL 입력, 월간 세션 수 입력 (현재 실제 수치 또는 예상치)
  • 사이트 소유권 확인: DNS 방식 또는 WordPress 플러그인 중 선택
  • 우리 서비스는 Cloudflare + DNS 방식 사용 권장
2

Cloudflare 연동 (권장 방법)

Ezoic 대시보드 → Site Settings → Nameservers → Cloudflare Integration 선택

  • Cloudflare에서 Ezoic IP를 A 레코드에 추가하는 방식
  • 기존 Cloudflare Workers 설정과 충돌 없이 동작
  • Ezoic이 CDN 앞단에 위치하여 광고를 자동으로 삽입
ℹ️ Cloudflare Pages 배포를 유지하면서 Ezoic을 함께 쓰려면 Cloudflare Worker를 통한 Ezoic 통합 문서를 참고하세요:
→ Ezoic Dashboard → Settings → Cloudflare → "Add via Worker"
3

ads.txt 업데이트

Ezoic 승인 후 제공되는 Publisher ID를 public/ads.txt에 추가합니다.

📄 public/ads.txt
google.com, pub-4166976105261105, DIRECT, f08c47fec0942fa0
- # ezoic.com, XXXXXXXXX, DIRECT
+ ezoic.com, YOUR_EZOIC_PUBLISHER_ID, DIRECT
+ (Ezoic이 추가로 요청하는 라인들을 그대로 붙여넣기)
⚠️ Ezoic 대시보드의 Monetization → ads.txt 탭에서 복사해야 할 전체 라인이 제공됩니다. 그 내용을 그대로 사용하세요.
4

Ezoic EPMV 대시보드 확인

통합 완료 후 48~72시간 이내에 Ezoic이 사이트를 분석하고 광고 최적화를 시작합니다.

  • Ezoic Big Data Analytics로 RPM/EPMV 모니터링
  • Ad Tester로 광고 위치/개수 A/B 테스트 자동화
  • Leap로 Core Web Vitals 자동 최적화
✅ Ezoic 전환 후 기존 AdSense 코드는 유지해도 됩니다. Ezoic이 AdSense를 포함한 여러 광고 네트워크를 통합 관리합니다.
✅ 완료 체크리스트
