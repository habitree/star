# 운세 표현 라이브러리 (Horoscope Phrase Library)

## 개요

이 라이브러리는 운세 콘텐츠를 자동 생성할 때 사용하는 재활용 가능한 문장 패턴과 표현 모음입니다.
각 표현은 `{행성}`, `{별자리}`, `{영역}`, `{카테고리}`, `{원소}`, `{계절}`, `{요일}`, `{특성}`, `{시간대}`, `{구체적 활동}` 등의 템플릿 변수를 포함할 수 있으며, 런타임에 실제 값으로 치환됩니다.

**사용 가능한 행성:** 태양, 달, 수성, 금성, 화성, 목성, 토성, 천왕성, 해왕성, 명왕성
**원소:** 불(Fire), 흙(Earth), 공기(Air), 물(Water)
**카테고리:** 종합운, 사랑운, 직업운, 건강운, 금전운

## 1. 도입부 패턴 (Opening Patterns) - 50가지

### 행성 에너지 도입 (15가지)

1. "오늘 {행성}의 에너지가 당신의 {영역}에 활력을 불어넣습니다."
2. "{행성}이(가) {별자리}를 비추며 새로운 가능성을 열어줍니다."
3. "{행성}의 순행이 {카테고리}에 긍정적 영향을 미칩니다."
4. "{행성}과 {행성}의 조화로운 배치가 오늘 하루를 빛나게 합니다."
5. "{행성}이(가) 당신의 별자리 영역으로 이동하면서 변화의 바람이 불어옵니다."
6. "강렬한 {행성}의 기운이 {카테고리} 전반에 걸쳐 영향력을 발휘합니다."
7. "{행성}의 빛이 {별자리}의 내면 깊숙한 곳까지 비추고 있습니다."
8. "오늘 {행성}이(가) 보내는 신호에 귀를 기울여 보세요."
9. "{행성}의 위치가 당신에게 유리한 방향으로 정렬되고 있습니다."
10. "{행성}과 {행성} 사이의 에너지 교류가 활발한 날입니다."
11. "{행성}이(가) 역행을 마치고 본래의 힘을 되찾고 있습니다."
12. "오늘 {행성}의 파동이 {영역}을(를) 중심으로 퍼져나갑니다."
13. "{행성}의 축복 아래, {카테고리}에서 주목할 만한 흐름이 감지됩니다."
14. "{행성}이(가) 당신의 운명의 교차점에 서 있습니다."
15. "우주가 {행성}을 통해 전하는 메시지가 있습니다."

### 시간/계절 도입 (10가지)

1. "오늘의 별자리 에너지는 {계절}의 {특성}과 어우러집니다."
2. "{요일}의 기운이 {원소} 별자리에게 특별한 메시지를 전합니다."
3. "이번 주의 천체 흐름이 절정에 다다르는 하루입니다."
4. "새로운 한 주의 시작과 함께 별들의 배치도 변화를 맞이합니다."
5. "{계절}의 전환기에 놓인 오늘, 하늘이 보내는 기운에 주목하세요."
6. "한 달의 중반, 축적된 에너지가 드러나기 시작하는 때입니다."
7. "오늘의 천체 배열은 {계절} 특유의 깊은 에너지를 반영합니다."
8. "{요일}마다 찾아오는 {행성}의 고유한 리듬이 오늘도 함께합니다."
9. "밤하늘의 별자리가 계절의 변화와 함께 새로운 이야기를 시작합니다."
10. "시간의 흐름 속에서 오늘이라는 날은 특별한 의미를 품고 있습니다."

### 감성적 도입 (15가지)

### 상태 묘사 도입 (10가지)

## 2. 마무리/조언 패턴 (Closing/Advice Patterns) - 50가지

5. "우주가 당신의 편이라는 사실을 잊지 마세요."
6. "씨앗은 이미 뿌려졌습니다. 곧 아름다운 꽃이 필 것입니다."
7. "두려움 뒤에 숨어 있는 기회를 놓치지 마세요."
8. "오늘 당신이 내딛는 한 걸음이 세계를 바꿀 수 있습니다."
9. "별들은 준비된 자에게 가장 밝은 빛을 보냅니다."

## 부록: 사용 가이드

### 조합 규칙

운세 문장은 다음 구조로 조합합니다:

```
[도입부 패턴] + [연결어] + [카테고리별 표현] + [원소별 수식어] + [마무리 패턴]
```

### 템플릿 변수 치환 규칙

| 변수 | 설명 | 예시 값 |
|------|------|---------|
| `{행성}` | 영향을 미치는 천체 | 태양, 달, 수성, 금성, 화성, 목성, 토성, 천왕성, 해왕성, 명왕성 |
| `{별자리}` | 대상 별자리 | 양자리, 황소자리, 쌍둥이자리, 게자리, 사자자리, 처녀자리, 천칭자리, 전갈자리, 사수자리, 염소자리, 물병자리, 물고기자리 |
| `{영역}` | 삶의 영역 | 사랑, 직업, 건강, 재정, 가정, 인간관계, 자아실현 |
| `{카테고리}` | 운세 카테고리 | 종합운, 사랑운, 직업운, 건강운, 금전운 |
| `{원소}` | 별자리 원소 | 불, 흙, 공기, 물 |
| `{계절}` | 현재 계절 | 봄, 여름, 가을, 겨울 |
| `{요일}` | 현재 요일 | 월요일, 화요일, 수요일, 목요일, 금요일, 토요일, 일요일 |
| `{특성}` | 계절/시기의 속성 | 생명력, 풍요로움, 수확, 성찰 |
| `{시간대}` | 하루 중 시간대 | 아침, 오전, 점심, 오후, 저녁, 밤 |
| `{구체적 활동}` | 권장 활동 | 독서, 산책, 명상, 요리, 정리정돈, 운동 |

### 톤/무드 매칭 가이드

| 운세 점수 범위 | 권장 톤 | 사용 표현 유형 |
|----------------|---------|----------------|
| 80~100 | 밝고 적극적 | 긍정적 표현 + 격려형 마무리 |
| 60~79 | 따뜻하고 안정적 | 긍정적/중립적 표현 + 격려형 마무리 |
| 40~59 | 차분하고 균형적 | 중립적 표현 + 행동 제안형 마무리 |
| 20~39 | 조심스럽고 배려적 | 중립적/주의 표현 + 주의형 마무리 |
| 0~19 | 신중하고 보호적 | 주의 표현 + 주의형 마무리 + 격려 1문장 추가 |
